- **Purple**: `#9333EA` (purple-600)

- **Definition**: Blue-50 bg, Blue-500 border

- **Warning**: Orange-50 bg, Orange-300 border

- **Success**: Green-500, Green-600
- **Error**: Red-500, Red-600
- **Warning**: Orange-500, Yellow-500
